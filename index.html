CSS Selectors Cheat Sheet

Selector Syntax Example What is it used for?
ID Selector #idName <div id="address"> #address {} To select and format a single element that have the given unique ID in a document.
Class Selector .className <div class="address"> .address {} To select and format all elements that have the given value of the class attribute.
Element Selector elementType <div> div {} To select and format HTML elements based on the element name
Attribute Selector type[attribute] <div style="color:yellow"> div[style] {} To select and format an element(s) with a specific attribute or attribute value.
Descendant Selector A B <div> <p> </p> </div> div p {} To select and format an element that is a descendant of another element
Child Selector A > B <div> <p> </p> </div> div > p {} To select and format a direct child element(s)
Multiple Selectors A, B <h1></h1> <p></p> h1, p {} To select and format all specified elements in a single CSS rule
Universal Selector * * {} To select and format all elements unless specified
